React and tosijs

A lot of developers are going to be using React. Is tosijs relevant to React developers?

Absolutely! The tiny react-tosijs library provides:

  • useTosi() to create hooks that connect any data into a React component.
  • the reactWebComponents proxy that turn any web-component into a React functional component.
