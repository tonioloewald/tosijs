Migrating from xinjs to tosijs

In a nutshell:

  1. Update to xinjs (and xinjs-ui) 1.0.6
  2. Fix any issues
  3. Replace all references to "xinjs" with "tosijs"

xinjs and tosijs 1.0.6 should be identical (likewise xinjs-ui and tosijs-ui), so the only thing you need to change should be the module names.

Please let me know if there are any issues.
